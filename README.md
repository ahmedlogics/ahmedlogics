## 💼 Featured Projects

### 🛡️ AI RAG Security Threat Prioritizer
An SOC-style system built to prioritize real cyber threats and filter alert noise.  
✅ Built using **Python, Streamlit, LangChain,** and **ChromaDB** ✅ Features a Retrieval-Augmented Generation (RAG) pipeline to fetch verified CVE mitigations.  
✅ Reduced analyst response time to <1s with citation-backed analysis.  
🏆 *Awarded “Most Creative Project” at Engineer’s Code 2025.*

### 📱 Neighbourly (Local Service Super-App)
A dual-sided service and P2P rental marketplace with real-time geospatial matching.  
✅ Built using **React Native, Firebase,** and **Zustand** ✅ Integrates Easypaisa/JazzCash APIs into a secure Escrow payment gateway.  
✅ Features a custom "Voice-Note-First" chat to eliminate literacy barriers.

### 🚀 Tanker Mafia Management System (TMMS)
A full-stack platform digitizing and streamlining urban water tanker operations.  
✅ Built using **TSX (React), Python (Flask),** and **Oracle SQL** ✅ Role-based access with fixed pricing models and area-based booking limits.  
✅ Admin analytics, live order tracking, and integrated feedback mechanisms.

### ☀️ SolarEase – Solar Energy Platform
E-commerce and service platform covering the entire solar lifecycle.  
✅ Built using **React.js, Node.js (Express),** and **MongoDB** ✅ Facilitates installment purchases, installation scheduling, and maintenance.  
✅ Secured with JWT-based authentication and robust Role-Based Access Control (RBAC).
